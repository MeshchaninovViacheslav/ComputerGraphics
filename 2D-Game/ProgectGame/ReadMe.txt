Список реализованных пунктов из базовой части:
1)  Чтение карты лабиринта из текстового файла.
    Площадь карты составляет 2500 тайлов, видимая площадь - 1600 тайлов.

2)  Визуализация карты при помощи графики с обязательным отображением всех игровых элементов разными изображениями.
    Вода(пустое пространство), стены, пол, игрок, выход с уровня, двери.

3)  Реализация движения и взаимодействия игрока с окружением при помощи клавиш W, A, S, D, E.
    Игрок не проходит сквозь стены.

4)  При попадании в воду игрок умирает: с помощью графики выводится сообщение о проигрыше и игра завершается.

5)  Если игрок достиг выхода с уровня, выводится при помощи графики сообщение о переходе на следующий уровень,
    либо об успешном окончании игры, если это был последний уровень.

Список реализованных пунктов из Дополнительной части:
1)  Анимация статических объектов: перетекание воды, возгорание огоньков на стенах.

2)  Реализован графический эффект перехода между уровнями - Fade.

3)  Реализованы источники света - лампы, реализована анимация источников света.

4) Реализован большой лабиринт - 2500 тайлов, видимая область - 1600 тайлов.
